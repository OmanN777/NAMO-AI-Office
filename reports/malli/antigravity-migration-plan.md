# MIGRATION PLAN: GEMINI CLI ➡️ ANTIGRAVITY CLI
**รายงานโดย:** Malli (เลขาบริหาร)
**วันที่ประกาศ:** 21 พฤษภาคม 2026
**เส้นตายการย้ายระบบ:** 18 มิถุนายน 2026 (เหลือเวลา 28 วัน)

---

## ⚠️ สรุปสถานการณ์ (The "Antigravity" Shift)
Google ได้ประกาศเปลี่ยนผ่านจาก Gemini CLI ไปสู่ **Antigravity CLI** ซึ่งเป็นแพลตฟอร์มใหม่ที่เน้นการทำงานร่วมกันของ Agent หลายตัว (Multi-Agent Orchestration) และมีความเร็วสูงกว่าเดิม (เขียนด้วยภาษา Go) ค่ะ

---

## 🛠️ ผลกระทบต่อ "Namo Executive Office"
1.  **Service Interruption:** หากไม่ย้ายระบบภายในวันที่ 18 มิ.ย. ทีม Agent ของเรา (Oman, มะลิ, Scout ฯลฯ) จะ **ไม่สามารถรับส่งข้อมูล** กับโมเดล Pro/Ultra ได้ค่ะ
2.  **Compatibility:** ฟีเจอร์หลักอย่าง **Agent Skills, Subagents และ Memory** จะถูกเปลี่ยนชื่อเป็น "Plugins" แต่ยังสามารถใช้งานต่อได้ค่ะ
3.  **Performance:** เราจะได้ระบบที่ทำงานแบบ Asynchronous ได้ดีขึ้น (Oman สามารถวิเคราะห์หุ้นในพื้นหลังขณะที่เราสั่งงานอื่นได้)

---

## 📅 Roadmap การย้ายระบบ (Proposed Plan)
### **Phase 1: Readiness (สัปดาห์นี้)**
- [ ] สำรองข้อมูล (Backup) โฟลเดอร์ `.gemini/`, `briefs/`, `portfolio/` และ `memory/` ทั้งหมด
- [ ] ตรวจสอบความพร้อมของสภาพแวดล้อม (Go Runtime) สำหรับติดตั้ง Antigravity CLI

### **Phase 2: Shadow Transition (ต้นเดือนมิถุนายน)**
- [ ] ติดตั้ง Antigravity CLI ควบคู่กับระบบเดิม
- [ ] ทดสอบการรัน Agent Oman ในระบบใหม่ (Dual-mode) เพื่อเช็คความแม่นยำของข้อมูล

### **Phase 3: Final Switch (ก่อน 15 มิถุนายน)**
- [ ] ย้ายไฟล์คอนฟิกและ Skills ทั้งหมดเข้าสู่โครงสร้าง Plugin ของ Antigravity
- [ ] ปิดการใช้งาน Gemini CLI อย่างเป็นทางการ
