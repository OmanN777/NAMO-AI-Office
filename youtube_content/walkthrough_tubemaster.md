# 🎬 Walkthrough: The TubeMaster Agent

มอลิได้ทำการติดตั้งและเทรนงานให้กับ **TubeMaster** (ผู้กำกับคลิป YouTube ส่วนตัวของบอส) เรียบร้อยแล้ว 100% ครับ! นี่คือสรุปความสามารถใหม่ของออฟฟิศเรา:

## 1. TubeMaster Protocol (SOP)
เราได้สร้างระบบการคิดงานสำหรับ TubeMaster โดยเฉพาะ ซึ่งเชี่ยวชาญด้าน:
- **Viral Hook Engineering:** เขียนบท 3 วินาทีแรกให้ดึงดูด
- **Visual Directing:** ร่าง Prompt ภาษาอังกฤษให้บอสนำไปเจนภาพใน Midjourney ได้ทันที
- **Metadata:** คิด Title, Tags และ Description

*(ไฟล์ SOP ถูกจัดเก็บไว้ที่: `plugins/namo-executive/skills/tube-master/SKILL.md`)*

## 2. Mascot Memory (ฝังความจำ "นายไบต์")
มอลิได้นำประวัติและนิสัยของ **"นายไบต์ (Mr. Byte)"** ฝังลงไปในฐานข้อมูล SQLite (หมวด `BrandIdentity`) เรียบร้อยแล้ว 
- **นิสัย:** เด็กไอทีหัวฟู ใต้ตาคล้ำ หน้ามึน
- **จุดเด่น:** มักจะพาตัวเองไปอยู่ในสถานการณ์วินาศสันตะโรทางเทคโนโลยีเสมอ
การฝังข้อมูลนี้จะทำให้ทุกครั้งที่บอสสั่ง TubeMaster เขียนสคริปต์ ตัวละครนายไบต์จะออกมามีนิสัยคงเดิมเป๊ะๆ ครับ!

## 3. การทำงานร่วมกับ Scout (สายสืบ)
ต่อไปนี้ บอสสามารถสั่งแบบคอมโบได้เลย เช่น: 
> *"ให้ Scout ไปกวาดข่าวไอทีล่าสุด แล้วส่งให้ TubeMaster แต่งเป็นสคริปต์นายไบต์ให้หน่อย"*
